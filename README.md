# MP 스마트 사냥 봇 (MP YOLO Bot Ultimate)

**와상장애 및 안구 마우스 사용자를 위해 특수 제작된 MP(구 메이플플래닛) 전용 인공지능 사냥 자동화 프로그램입니다.**

드래그(Drag) 조작이 불가능한 안구 마우스 환경의 한계를 극복하기 위해, **클릭(눈 깜빡임) 한 번만으로 AI를 학습시키고, 사냥터의 동선(노드)을 눈으로 찍어서 그릴 수 있는 맵 에디터 기반의 완벽한 자동 사냥 환경**을 제공합니다.

---

## ✨ 봇의 진화된 주요 기능 (Ultimate Features)

### 1. 통합 메인 GUI (제어반)
- 맵 설정 파일과 인공지능 가중치 모델을 콤보박스로 쉽게 선택하여 교체할 수 있습니다.
- 사냥을 시작하고 중지하는 직관적이고 거대한 버튼을 제공합니다.

### 2. 닉네임 정밀 추적 기능 (Nickname Tracking)
- 캐릭터가 항상 화면 중앙에 있지 않더라도, 화면에서 내 **닉네임 이미지**를 실시간으로 찾아내어 캐릭터의 오차 없는 (X, Y) 좌표를 획득합니다.
- 이를 통해 어떤 해상도나 카메라 이동 속에서도 정확한 몬스터 타겟팅이 가능합니다.

### 3. 미니맵 액션 노드 에디터 (Map Editor)
- 드래그 없이 미니맵을 캡처한 뒤 원하는 곳을 **눈으로 한 번만 클릭**하여 행동 구역을 설정합니다.
- 좌/우 걷기, 좌/우 점프, 줄타기, 밑점프 등 원하는 행동을 구역마다 매핑하고 맵 파일(`.json`)로 저장할 수 있습니다.

### 4. 커스텀 키 세팅 & 세부 전투 설정
- **키바인딩**: 점프, 공격, 텔레포트 스킬 키를 내 맘대로 영어로 적어(예: `ctrl`, `shift`) 지정할 수 있습니다.
- **점프 공격 / 텔레포트 이동**: 표창/단검 도적을 위한 공격 전 점프 모드, 마법사를 위한 이동 시 텔레포트 연사 모드를 지원합니다.
- **몬스터 밀집도 조건**: 캐릭터 기준 X, Y 반경 내에 몹이 'N마리 이상' 있을 때만 공격하게 설정하여 마나 낭비를 줄입니다.

## 🚀 설치 방법 (Installation)

1. 파이썬(Python 3.8 이상)이 설치되어 있어야 합니다.
2. 이 저장소를 다운로드 받거나 클론(Clone)합니다.
3. 명령 프롬프트(CMD)를 열고 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

## 🛠️ 사용 가이드 (How to Use)

### [학습 모드] 몬스터 AI 가르치기
1. `python data_collector.py` 실행: 사냥터 스크린샷 100장을 모읍니다.
2. `python one_click_labeler.py` 실행: 수집된 사진 속 몬스터의 정중앙을 안구 마우스로 딱 1번만 콕콕 클릭합니다. 몬스터 종류가 달라도 상관없이 닥치는 대로 다 클릭하면 됩니다.
3. `python train.py` 실행: 몬스터를 학습하여 `best.pt` 파일을 만듭니다.

### [실전 모드] 사냥터 맵(동선) 그리고 자동사냥 켜기
1. `python map_editor.py` (혹은 메인 UI에서 에디터 열기)를 켜서 캡처된 미니맵 위에 길을 뚫어줍니다. (예: 발판 끝에는 [우측 점프], 밧줄 아래엔 [줄타기])
2. 맵을 저장한 후 `python main.py`를 켭니다.
3. 화면 속 내 닉네임을 캡처해두고, 맵과 가중치를 고른 뒤, 키 세팅을 점검합니다.
4. **[▶ 사냥 시작]** 버튼을 누르면 인공지능이 노드(길)를 따라 움직이며 설정된 마릿수 이상이 모였을 때 자동으로 공격합니다!

### [특수 기능] 거탐(거짓말 탐지기) 자동 해제 모듈 (`captcha_solver.py`)
거탐 팝업 발생 시, 딥러닝 학습(YOLO) 없이도 투명해지는 도형을 물고 늘어지는 **CSRT (Channel and Spatial Reliability Tracker)** 기반 실시간 추적 알고리즘을 사용합니다.
- **알고리즘 상세 작동 원리:**
  1. **초기 스캔 (Template Matching)**: 팝업이 뜨고 최초 10초 이내에 도형이 투명해지기 전(선명할 때) 별, 원, 세모 모양을 `cv2.matchTemplate`을 사용해 화면에서 빠르게 스캔합니다.
  2. **초기 위치 획득 (Bounding Box)**: 도형이 감지되면 해당 영역 주변 50x50 픽셀의 초기 바운딩 박스를 계산합니다.
  3. **CSRT Tracker 초기화**: OpenCV의 `TrackerCSRT_create()`를 호출하여 스캔한 바운딩 박스 영역을 추적 목표(Lock-on)로 지정합니다. CSRT는 변형, 크기 변화, 배경 노이즈(스크롤되는 돌바닥) 환경에서도 픽셀의 채널별 신뢰도와 공간적 신뢰도를 계산하여 형태를 끝까지 추적하는 가장 강력한 고전 알고리즘입니다.
  4. **마우스 밀착 추적**: 이후 도형이 투명해지고 배경이 울렁거리더라도, Tracker가 계산해 내는 도형의 중심 좌표를 향해 `0.05초(20 FPS)` 간격으로 마우스 포인터를 끊임없이 이동(Tracking)시켜 도형이 사라질 때까지 완벽하게 밀착 마크합니다.
