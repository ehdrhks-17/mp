# 메이플플래닛 스마트 사냥 봇 (MaplePlanet YOLO Bot)

**와상장애 및 안구 마우스 사용자를 위해 특수 제작된 메이플스토리(메이플플래닛) 전용 인공지능 사냥 자동화 프로그램입니다.**

드래그(Drag) 조작이 불가능한 안구 마우스 환경의 한계를 극복하기 위해, **복잡한 라벨링 과정 없이 클릭(눈 깜빡임) 한 번만으로 인공지능(YOLOv8)을 학습시키고 자동 사냥을 수행**할 수 있도록 설계되었습니다.

---

## ✨ 주요 기능 (Features)

1. **안구 마우스 전용 거대 UI (Eye-Mouse Friendly UI)**
   - 눈으로 클릭하기 쉽도록 화면 우측 하단에 거대한 [시작/정지] 토글 버튼을 제공합니다.
   - UI 클릭 시 게임 창의 포커스를 뺏지 않아 매끄러운 조작이 가능합니다.

2. **드래그 없는 원클릭 AI 라벨링 (One-Click Labeling)**
   - 머신러닝 학습을 위한 정답 박스(Bounding Box)를 그릴 때 마우스 드래그가 전혀 필요하지 않습니다.
   - 사진에 찍힌 몬스터의 정중앙을 안구 마우스로 딱 한 번만 클릭하면 자동으로 라벨링이 완료됩니다.

3. **YOLOv8 기반 스마트 몬스터 인식 및 추적 사냥**
   - 학습된 인공지능이 화면 상의 몬스터를 실시간으로 인식하고, 해당 방향으로 이동하여 공격합니다.
   - 화면에 몬스터가 없을 경우, 벽에 닿을 때까지 자동으로 방향을 전환하며 맵을 순찰(Blind Patrol)합니다.

*(참고: 타 유저 감지 시 사냥을 정지하는 기능은 사용자 요청에 의해 현재 코드에서 제외되었습니다.)*

## 🚀 설치 방법 (Installation)

1. 파이썬(Python 3.8 이상)이 설치되어 있어야 합니다.
2. 이 저장소를 다운로드 받거나 클론(Clone)합니다.
3. 명령 프롬프트(CMD)를 열고 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

## 🛠️ 사용 방법 (How to Use)

본 프로그램은 **[나만의 AI 만들기]** 단계와 **[사냥 봇 실행]** 단계로 나뉩니다.

### 1단계: 몬스터 사진 수집하기 (`data_collector.py`)
1. 메이플플래닛을 켜고 사냥터에 입장합니다.
2. `python data_collector.py`를 실행합니다.
3. 2초마다 알아서 게임 화면이 캡처되어 `dataset/images/train/` 폴더에 저장됩니다. 
4. 몬스터의 다양한 모습이 찍히도록 약 50~100장 정도 모이면 스크립트 창에서 `Ctrl+C`를 눌러 종료합니다.

### 2단계: 원클릭으로 몬스터 정답 알려주기 (`one_click_labeler.py`)
1. `python one_click_labeler.py`를 실행합니다.
2. 수집된 사진들이 화면에 크게 뜹니다.
3. **사진 속 몬스터의 정중앙을 안구 마우스로 딱 한 번 클릭(깜빡)하세요.**
4. 자동으로 네모 박스가 저장되며 다음 사진으로 넘어갑니다. 허공 사진이면 우측 하단의 [건너뛰기]를 누르세요.

### 3단계: 인공지능 버튼 하나로 학습시키기 (`train.py`)
1. `python train.py`를 실행합니다.
2. YOLOv8 엔진이 수집된 사진과 정답을 바탕으로 몬스터의 생김새를 학습합니다. (몇 분 소요)
3. 완료되면 봇 폴더 안에 `best.pt` 라는 가중치(인공지능) 파일이 짠 하고 나타납니다!

### 4단계: 사냥 봇 가동하기 (`main.py`)
1. `python main.py`를 실행합니다.
2. 화면 구석에 반투명한 거대 UI가 나타납니다.
3. 안구 마우스로 **[▶ 사냥 시작]** 버튼을 클릭하면 캐릭터가 `best.pt`를 활용해 스스로 사냥을 시작합니다!

## ⌨️ 단축키 및 환경 설정 수정
점프나 공격 키를 바꾸고 싶으시다면 `controller.py` 파일을 메모장으로 여신 후 `KEYS` 딕셔너리 부분을 수정하시면 됩니다. 기본값은 점프(Alt), 공격(Ctrl)으로 세팅되어 있습니다.
